# Portafolio Personal

Sitio sencillo para mostrar mi experiencia, proyectos y formas de contacto.

## Cómo usarlo
- Instalar dependencias: `npm install`
- Iniciar en modo desarrollo: `npm run dev`
- Construir para producción: `npm run build`

## Contenido
- Inicio y presentación
- Línea de tiempo de experiencia
- Enlaces a redes y contacto
